# C# Port (Prototype)

This directory contains an early attempt to port parts of the PHP Laravel
application to C# using ASP.NET Core. It only demonstrates the
structure of the future application.

## Structure
- `Program.cs` – entry point creating a basic web host.
- `Controllers/HomeController.cs` – sample controller returning a placeholder
  message.

## Usage
This code has not been wired into the existing PHP project. It merely shows
how a C# version could start. Additional controllers, models and views would
need to be implemented to reach parity with the Laravel project.
